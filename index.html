Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The lang="en" attribute tells the browser the language of the webpage.
  • The charset element sets the character encoding.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a website.
  • The link element connects an external source to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.

CSS

  • An inline CSS can style an element directly in the HTML file.
  • An internal style sheet can embed an entire stylesheet directly in the HTML file.
  • An external css style sheet can keep all the CSS rules in a separate file.
  • The margin property indicates the amount of space around the outside of an element.
  • The padding property indicates the space around the content inside the element.
  • The * wild card symbol applies a rule to all elements visible on the page.
  • The img element applies rules to any image element in the markup.
  • A class attribute allows us to share a CSS rule to any element we choose by assinging the rule to a class attribute with a class selector.
  • Any HTML element assigned to a class will gain those property assignments.
  • CSS can be used to assign borders, box-shadows, color, font size, and other stylistic choices.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: receive a branch's modifications to the main branch
  • git push origin feature: push the changes locally to the remote branch
  • git status: check what branch you're on
  • git config: change the name associated with your commits
